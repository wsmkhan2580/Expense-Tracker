Mission 2 - Level 3 Project
📌 Overview
This is my Mission 2 Level 3 project — CashFlow Salary & Expense Tracker, built with Vanilla JavaScript (ES6 modules).
It tracks monthly salary vs. expenses in real time, shows a live Chart.js doughnut chart, and exports styled PDF reports with currency conversion.
The project demonstrates advanced frontend architecture and recruiter‑friendly modular design.

🚀 Features

✅ Salary input & persistent storage

✅ Add / delete expenses with instant balance updates

✅ Chart.js doughnut chart (Expenses vs. Remaining)

✅ PDF export with jsPDF (dark theme)

✅ Currency conversion via Frankfurter API

✅ Budget alert when balance <10%

✅ Smooth UI animations & keyboard shortcuts
